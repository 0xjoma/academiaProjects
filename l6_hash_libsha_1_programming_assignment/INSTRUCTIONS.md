## Prerequisites

- Completed the written exercises
- Read the NIST documents

---

## Programming Details

### Part 1 - HashLib (10 points)

Use the [hashlib](https://docs.python.org/3/library/hashlib.html) library to write a program that uses functions from hashlib. Use the following hash functions to hash your name:

- SHA-1
- SHA-256
- SHA-512
- SHA3-256

### Part 2 - SHA-1 Algorithm (100 points)

Program the SHA-1 algorithm. Note that SHA-1 is no longer a secure hash function; however, the SHA-2 algorithms operate in a similar way and are considered secure.

- The `SHA1.py` file has comments embedded throughout the code to assist in developing the SHA-1 algorithm and indicate where to add code.
- Add code under the comments that say **"add code here..."**
- If you encounter challenges, reach out to the instructor.

The exercise should be completed using **Python** or **Java**\*.

> \*Alternative Programming Option: If you prefer Java, you may use it, but helper files are only supplied in Python. Java programming tools are not available in the Virtual Lab environment.
